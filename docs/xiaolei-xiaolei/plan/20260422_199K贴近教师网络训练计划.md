# 20260422 199K 贴近教师网络训练计划

## 1. 这份计划回答什么问题

当前已经确认：

- `79 / 99 / 119` continuation seed 全部失败，没有一条超过 `119.pt`
- 单条和批量 `logit / prefix` 图表明，学生并没有把老师的输出行为学到位
- 重复 hard case 的流式实验又进一步说明：老师在长序列下可以稳定多次触发，学生不行

因此今天这份计划只回答一个问题：

- 下一步怎样让 `199K` 学生网络更接近老师网络

这里的“更接近老师”不是指：

- 机械地学出更多 `<filler>`

而是指：

- 更接近老师的输出分布
- 更接近老师的 blank 到 non-blank 过渡
- 更接近老师的 prefix/path 保持能力
- 更接近老师的多次流式触发稳定性

## 2. 当前证据带来的约束

### 2.1 不再沿当前 continuation 继续试

这条结论已经成立：

- 当前 `FA + CTC continuation` 路线阶段性收口
- 后续不再继续做新的 from79 / from99 / from119 / `v2` 分支

### 2.2 不把“每段都学出 filler”当成训练目标

重复 hard case `x3` 实验已经说明：

- 老师流式能稳定 `3/3` 触发
- 但老师并不会在每段前都重新给出一个强 `<filler>`

所以后续不能把训练目标写成：

- 让学生每段都模仿一个 `<filler>` 峰

更合理的目标是：

- 让学生更像老师那样保住前缀起步
- 让学生更像老师那样保住重复结构
- 让学生在长序列流式场景下稳定多次触发

## 3. 下一步训练主线

### 3.1 基线固定

后续所有“贴近老师”的训练实验，统一以：

- `distill 199K 119.pt`

作为当前最好学生起点。

### 3.2 第一优先级：output-level distillation

当前最值得先补的不是更多 continuation，而是：

- 在现有 block-level MSE 之外，再加 teacher-student 输出分布约束

原因很直接：

- 现在的蒸馏训练主要对齐的是中间 block 特征
- 老师 logits / probs / prefix 行为本身并没有被直接蒸馏
- 这也是为什么学生容易出现“中间特征看起来没完全坏，但输出路径明显不像老师”的现象

建议方向：

1. 在 finetune 阶段加入 teacher-student `KL` distillation
2. 不直接对 raw logits 做绝对值回归
3. 直接对 softmax 后的输出分布做约束

训练目标应更像：

- `CTC(student label) + block feature MSE + output KL`

### 3.3 第二优先级：blank / non-blank distillation

从图上看，学生一个核心问题是：

- 前半段太容易一直停在 `<blk>` 主导状态

因此比“给 `小` 单字加权”更合理的方向是：

- 增加一个 blank-vs-nonblank 层面的辅助蒸馏

重点不是逼学生学出固定 `<filler>`，而是：

- 当老师已经开始从 `<blk>` 释放出来时，学生不要还一直锁死在 `<blk>`

### 3.4 第三优先级：hard positive replay 继续保留

`hard positive replay` 仍然保留，而且要和 output-level distillation 配合。

因为当前困难样本并不是泛泛的所有正样本，而是：

- teacher-hit / student-miss

这批样本正好是最该让学生逼近老师的地方。

### 3.5 第四优先级：prefix/path 约束

如果前几步还不够，再考虑更强的 prefix/path 约束。

更值得盯的是：

- `小`
- `小雷`
- `小雷小`
- `小雷小雷`

而不是只盯：

- 单个 `小`

## 4. 建议的实验顺序

当前建议按这个顺序推进：

1. 先补流式口径的 baseline 对照
   - `119.pt` 流式评测
   - `159.pt` 流式评测
   - 先把“原始 test split 上 teacher/student 的流式差距”定量化
2. 再补多条 repeated hard case
   - 看“第一段漏掉”是否稳定出现
3. 然后开始训练改动
   - 先上 output-level KL
   - 再上 blank / non-blank KD
   - 同时保留 hard positive replay
4. 每一版新训练都要同时过三道 gate
   - 原始 test split
   - 重复 hard case 的流式稳定性
   - 全量 streaming eval

## 5. 当前建议的成功标准

新的“贴近老师”训练，不只看原始 accuracy。

至少同时满足：

1. 原始 `test split` 的 `小雷小雷` 超过 `119.pt`
2. `小雷快拍` 不明显拉坏
3. repeated hard case 流式触发次数比 `119.pt` 更接近老师
4. 不把“每段都打出 filler”当成成功标准

## 6. 当前明确不建议做的事

- 不再继续扩 continuation 超参
- 不把“学会每段 filler”当成直接目标
- 不直接给 `小` 单字加权
- 不先上泛化噪声增强试图自动解决

## 7. 当前结论收口

接下来最合理的主线不是：

- 再找一个更好的 continuation checkpoint

而是：

- 开始让学生直接学习老师的输出行为

更准确地说：

- 从“feature alignment 蒸馏”走向“feature + output 行为蒸馏”
