# 小雷 0327 数据集生成方案

## 目标

先生成一版可直接用于 `fsmn + ctc` 2599 baseline 的训练数据，重点是把新 0327 数据和可选旧 hi/ni 语音统一整理成 `wav.scp / text / wav.dur / data.list`。

## 首轮数据范围

- 正样本：
  - `XIAOLEI_XIAOLEI`
  - `XIAOLEI_KUAIPAI`
- 负样本：
  - `FILLER`
  - 可选追加旧 `hi_xiaowen / nihaowenwen` 语音作为新任务 hard negative
- 暂不纳入训练：
  - `RAYNEO`

## 文本映射策略

为兼容当前 2599 CTC 词表，正样本直接落成真实字符序列：

- `<XIAOLEI_XIAOLEI>` -> `小 雷 小 雷`
- `<XIAOLEI_KUAIPAI>` -> `小 雷 快 拍`
- `<FILLER>` -> `<filler>`

说明：

- 现有 2599 词表已经包含 `小 / 雷 / 快 / 拍`，不需要扩词表。
- 第一版允许 `FILLER` 先保留为 `<filler>`，后续再用 ASR 对新负样本做二次筛洗。
- 旧 `hi_xiaowen / nihaowenwen` 语音只要不含新关键词，都可以作为新任务负样本。

## 切分策略

- 正样本：按说话人/录制组做 group split，避免同一人或同一录制组同时落到 train/dev/test。
- 负样本：随机抽样进入 train/dev/test。
- 首轮负样本比例：按 split 内 `neg : pos = 3 : 1` 控制。
- 默认切分：`train/dev/test = 90/5/5`。
