# 小雷小雷文档目录

本目录用于沉淀 `小雷小雷 / 小雷快拍` 项目的规划、实验、经验和常用命令。

目录约定：

- `plan/`：方案规划、任务拆解、阶段计划、设计决策。
- `know_how/`：经验总结、踩坑记录、数据规范、流程说明、复盘方法。
- `experiment/`：实验记录、结果分析、对比结论、版本回归。
- `常用命令_xlxl.txt`：和小雷数据集相关的训练、评测、蒸馏、转写、数据处理等单行命令。

命名约定：

- Markdown 文档默认使用 `YYYYMMDD_功能.md`
- 如需更细粒度区分，可使用 `YYYYMMDD_功能_补充说明.md`
- 常用命令按功能块分组，块内只保留可直接复制执行的单行命令

使用约定：

- 后续小雷项目的规划文档放到 `plan/`
- 后续小雷项目的 know-how 放到 `know_how/`
- 后续小雷项目的实验文档放到 `experiment/`
- 提到“存到常用命令里”时，先判断是否应写入 `常用命令_xlxl.txt`
